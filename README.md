# Superstore Sales Analysis 📊

## Objective
To analyse sales data of the corporate customer segment at Superstore to identify revenue trends, top performing categories and growth opportunities.

## Tools Used
- Microsoft Excel

## Key Analysis
- Identified top performing product categories by revenue
- Analysed sales trends across different regions
- Highlighted growth opportunities within corporate segment

## Key Findings
- Derived actionable insights to improve sales strategy
- Identified underperforming segments requiring attention

Data Analysis | Excel | Business Insights | Reporting
